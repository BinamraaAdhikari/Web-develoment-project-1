# Internship Tracker REST API

This project is a backend-focused REST API that allows users to securely track and manage their internship or job applications. 
It supports user authentication, CRUD operations, and relational data storage using PostgreSQL.


- User registration and login
- JWT-based authentication
- Create, read, update, and delete internship applications
- User-specific data access (authorization)
- PostgreSQL relational database
- Interactive API documentation using Swagger
